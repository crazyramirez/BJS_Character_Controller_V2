  2. Open terminal in project folder and run npm install
  3. Then run npm start
  4. Open http://localhost:3000/builder.html

Animation Events

Add gameplay markers for footsteps, hits, casts, sounds and particles. A marker fires when playback crosses its frame — preview an animation in the viewport to see markers fire live (toast + console). In your game, listen with charCtrl.onAnimationEvent = (evt, anim) => {...} or window.addEventListener('charanimevent', ...).
